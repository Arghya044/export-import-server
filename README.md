# Import Export Hub Server

A backend server for managing import and export operations, built with Express.js and MongoDB.

## Features

- **Product Management**: Create, read, update, and delete products
- **Import Operations**: Track and manage product imports
- **User Management**: Handle user registration and data
- **Search Functionality**: Search products by name
- **User-specific Data**: Retrieve exports and imports by user email
- **Latest Products**: Get the most recently added products

## Tech Stack

- **Node.js**: Runtime environment
- **Express.js**: Web framework
- **MongoDB**: Database
- **CORS**: Cross-origin resource sharing
- **dotenv**: Environment variable management
- **Vercel**: Deployment platform


### Users

- **POST** `/users` - Create or check if user exists

## Deployment

This project is configured for deployment on Vercel.
